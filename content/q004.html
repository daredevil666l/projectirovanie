4. Классификация информационных систем. Виды классификаций

Классификация ИС — это разбиение систем на группы по выбранным признакам. В презентации подчеркивается, что ИС можно классифицировать по разным основаниям, и на практике часто используют сразу несколько признаков одновременно.

Основные признаки (виды классификаций), встречающиеся в презентации:

по признаку структурированности решаемых задач;

по характеру представления и логической организации хранимой информации (тип данных);

по выполняемым функциям и решаемым задачам;

по характеру обработки информации на различных уровнях управления предприятием;

по уровням управления;

по функциональному признаку;

по сфере применения;

по степени распределенности (настольные/локальные и распределенные).

Отдельно на схеме показаны варианты классификации по типу данных, степени автоматизации, характеру обработки данных, уровню управления и сфере применения.

Слайд из презентации: обобщающая схема классификации ИС.

Возможные дополнительные вопросы и ответы

1) Какие два вида ИС выделены на схеме по типу данных?

Фактографические и документальные.

2) Какие виды ИС показаны по степени автоматизации?

Ручные, автоматизированные и автоматические.

3) Какие виды ИС выделяются по характеру обработки данных на схеме?

Информационно‑поисковые, информационно‑решающие, управляющие и советующие.

4) Какие виды ИС показаны по уровню управления на схеме?

Стратегические, функциональные и операционные.

5) Какие виды ИС приведены на схеме по сфере применения?

Интегрированные, организационного управления, управления ТП и САПР.
