# PWA экзамен-справочник (GitHub Pages)

## Как выложить на GitHub Pages
1) Создай репозиторий на GitHub.
2) Залей **все файлы из этого архива** в корень репозитория (или в папку `/docs`, если так удобнее).
3) В репозитории: Settings → Pages:
   - Source: Deploy from a branch
   - Branch: `main` / root (или `/docs` если выбрал)
4) Открой выданный GitHub Pages URL. На телефоне браузер предложит «Добавить на главный экран».

## Оффлайн-работа
Service Worker при первом запуске скачивает и кэширует **все** файлы (текст + изображения).
Дальше приложение открывается и без интернета.
